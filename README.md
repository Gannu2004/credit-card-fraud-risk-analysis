## Overview
Interactive Power BI dashboard for credit card fraud analytics.

## Features
- 5 KPI Cards: Fraud Rate, Fraudulent Transactions, Critical Risk, Fraud Amount, Top Fraud Type
- Bar Chart: Transaction Amount by Fraud Type & Category
- Donut Chart: Fraud Risk Breakdown (Critical/High/Medium/Low)
- Column Chart: Fraudulent Transactions by State
- Line Chart: Monthly Fraud Trends
- 3 Interactive Slicers: Fraud Type, State, Merchant Name

## Tools Used
Power BI Desktop | DAX | Data Modeling
